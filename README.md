# ⚡ Advanced Virtual Electrical Lab

An interactive web-based simulation platform to understand and visualize electrical circuits in real-time.

## 🚀 Overview
The **Advanced Virtual Electrical Lab** allows users to build and experiment with electrical circuits using components like batteries, resistors, bulbs (load), and fuses — all within a browser.

It focuses on making core electrical concepts intuitive through **real-time simulation + visual feedback**.

---

## 🎯 Objectives
- Simulate real-world electrical circuit behavior  
- Visualize relationships between voltage, current, and resistance  
- Demonstrate fuse protection and overload conditions  
- Provide an engaging and interactive learning experience  

---

## 🛠️ Tech Stack
- **HTML5** – Structure  
- **Tailwind CSS** – UI styling & responsiveness  
- **JavaScript** – Core logic & interactivity  
- **Canvas API** – Animated particle background  

---

## ✨ Features
- ➕ Add/remove electrical components dynamically  
- ⚙️ Adjustable voltage, resistance, load, and fuse rating  
- 📊 Real-time current calculation using Ohm’s Law  
- ⚡ Animated power flow visualization  
- 🔥 Fuse blow simulation on overload  
- 🔄 Reset circuit instantly  
- 🧾 Experiment history tracking  
- 🎨 Glassmorphism UI + animated starfield background  

---

## ⚙️ Working Principle
The system calculates total resistance based on selected components.  
When power is turned ON:
V=IR
- Current is calculated in real-time  
- If current exceeds fuse rating → fuse blows → circuit stops
